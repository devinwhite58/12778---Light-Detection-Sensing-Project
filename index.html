Proposed solution

From GridInSight Paper [1], we found out that different light sources have different wave characteristics. The way a bulb flickers, it can be related to the input voltage introduced as a unit-less Bulb Response Function (BRF). The BRF value at a time t is defined as the ratio of intensity measured to the temporal average of measured intensity. The major part of our project solution revolves around measuring a nominal BRF for a particular light source which is consistent to it.

How to measure the light intensity?

➢ Physical stimulus: The physical stimulus for our system as described above is light. We have different artificial light sources connected to AC power.

➢ Light intensity sensor: We use light sensors such as photoresistors, phototransistors and photodiodes to capture the intensity of each artificial light source.

➢ Analog signal conditioning: The photoresistor does not need any special conditioning circuit to receive an appropriate signal output. The sensor is simply connected directly to the RPi Pico W. The photodiode requires an op-amp or shunt resistance large enough to produce a voltage to be studied.

➢Signal Processing: Our signal processing includes the program that we wrote which took the raw signal output data. This data is available in the form of raw numbers that correlate to 16 bit numbers in our ADC and convert it to the voltage values by multiplying by 3.3 and 65535. We use the DSM trick to capture our data using fast sampling to utilize the existing portion of code. We then take these raw values and convert them into the frequency domain using an FFT module in NumPy. This helped us plot the light bulb response functions in the frequency domain. This will further aid us to differentiating the bulb types. The BRF’s obtained are then compared to the existing BRF’s to advice which light source is used.

➢ Signal Output: Signal output for out project is an FFT that is used to characterize the bulb.
